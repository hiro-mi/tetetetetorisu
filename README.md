# tetetetetorisu
HTML/CSS/JavaScript だけで作ったデスクトップ向けテトリス風クソゲーです。  
素朴に動きますが、挙動やスコア計算にムダなクセが大量に入っています。

## 遊び方
- `index.html` をブラウザで開くと即遊べます。インストール不要。
- 「スタート/リセット」でゲーム開始、`ポーズ` ボタンで一時停止・再開。
- 操作キー
  - ← →：左右移動
  - ↑：回転
  - ↓：ソフトドロップ
  - Space：ハードドロップ（落下距離に応じて減点）

## 入っているクソ仕様
- **スコア計算が逆張り**：まとめて消すほど大幅に減点され、単発消しが最も得。
- **操作逆転イベント**：落下完了後 25% の確率で上下左右の操作が 5 秒間逆転。ログに記録されます。
- **背景フラッシュ**：ラインを消すと背景が不快にチカチカします。
- **ハードドロップ減点**：早く終わらせようとするとスコアがどんどん下がります。
- **ムードメーター**：スコアとライン数に応じて顔文字が悪化していきます。

## 開発メモ
- 追加のクソ要素を入れる場合は `game.js` にコメントを残し、制御できるようにしてください。
- ビルドツールやライブラリは使っていません。静的ファイルだけで完結します。
